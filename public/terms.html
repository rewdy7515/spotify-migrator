Términos de Uso — Backup de Spotify

1. Función del servicio

Esta aplicación permite al usuario exportar y descargar información de su propia cuenta de Spotify para fines personales de respaldo (“backup”) y permite restaurar dicha información únicamente en la misma cuenta del usuario autenticado mediante OAuth de Spotify.

2. Uso personal

Toda la información exportada mediante esta aplicación es de uso estrictamente personal del usuario. La aplicación no facilita, promueve ni automatiza la transferencia de datos entre diferentes cuentas de Spotify.

3. Archivos cargados por el usuario

El usuario es responsable del archivo que decide cargar en la aplicación. La aplicación asume que el archivo subido pertenece al mismo usuario que está autenticado en Spotify. El origen o intercambio del archivo fuera de esta plataforma no es responsabilidad de la aplicación ni constituye una funcionalidad del servicio.

4. Prohibiciones

El usuario no debe utilizar esta aplicación para:
• Transferir u obtener datos de cuentas de terceros.
• Crear, compartir o distribuir respaldos de otras personas.
• Manipular cuentas a las cuales no tenga acceso autorizado mediante OAuth.

5. Relación con Spotify

Esta aplicación no está afiliada, patrocinada ni aprobada por Spotify AB. El usuario acepta cumplir con los Términos de Uso y Políticas de la API de Spotify.

6. Responsabilidad del usuario

El usuario asume toda responsabilidad por el uso que dé a los archivos exportados y por cualquier acción realizada fuera de esta aplicación. Cualquier intercambio de archivos entre personas es decisión exclusiva de los usuarios y no forma parte de las funcionalidades ofrecidas por el servicio.

7. Limitación de responsabilidad

La aplicación no será responsable por:
• El uso inapropiado de archivos exportados.
• Daños, pérdidas o accesos no autorizados que ocurran fuera de la plataforma.
• Acciones realizadas por terceros con los archivos generados.
